Pokedex Lookup Tool Documentation

Main Site

Get All Pokemon (/getAll, GET/HEAD)

Function

Retrieves every Pokemon in the database.

Parameters

No parameters supported.

Return Format

An array of JSON objects matching the data found in pokedex.json

Example

Request: /getAll

Result: [{'id': 1, 'num': '#001', 'name': 'Bulbasaur', 'type: ['Grass', 'Poison']', "height": "0.71 m", "weight": "6.9 kg", 'stats': {'hp': 45, 'atk': 49, ...}}, ...]

Get All Types (/getAllTypes, GET/HEAD)

Function

Retrieves every type in the database.

Parameters

No parameters supported.

Return Format

An array of JSON objects matching the data found in types.json

Example

Request: /getAllTypes

Result: [{"name": "normal", "weaknesses": ["fighting"], "resistances": [], "immunities": ["ghost"]}, ...]

Get Pokemon By Name (/getByName, GET/HEAD)

Function

Retrieves a Pokemon by its name.

Parameters

name (string, required): The name of the Pokemon you wish to retrieve.

Return Format

A JSON object matching the data found in pokedex.json

Example

Request: /getByName?name=bulbasaur

Result: {'id': 1, 'num': '#001', 'name': 'Bulbasaur', 'type: ['Grass', 'Poison']', "height": "0.71 m", "weight": "6.9 kg", 'stats': {'hp': 45, 'atk': 49, ...}}

Get Pokemon By Type (/getByType, GET/HEAD)

Function

Retrieves all Pokemon that match a specified amount of types.

Parameters

types (string, required): The type or types you wish to find Pokemon for. Sent as a comma-separated list

Return Format

An array of JSON objects matching the data found in pokedex.json

Example

Request: /getByType?types=normal,fire,water,

Result: [{"id": 4, "num": "004", "name": "Charmander", "img": "http://www.serebii.net/pokemongo/pokemon/004.png", "type": [ "Fire" ], "height": "0.61 m", "weight": "8.5 kg", "stats": { "hp": 39, "atk": 52, ...}}, ...]

Create Pokemon (/createPokemon, POST)

Function

Creates a Pokemon with the specified parameters.

Parameters

name (string, required): The name of your Pokemon

image Link (string, optional): A link to an image you want your Pokemon to have.

type 1 (string, required): The name of the first type of your Pokemon.

type 2 (string, optional): The name of the second type of your Pokemon.

height (m) (double, required) The height, in meters, of your Pokemon.

weight (kg) (double, required) The weight, in kilograms, of your Pokemon.

hp (int 1-255, required) The HP stat of your Pokemon

atk (int 1-255, required) The Attack stat of your Pokemon

def (int 1-255, required) The Defense stat of your Pokemon

spAtk (int 1-255, required) The Special Attack stat of your Pokemon

spDef (int 1-255, required) The Special Defense stat of your Pokemon

speed (int 1-255, required) The Speed stat of your Pokemon

Return Format

A JSON object matching the data found in pokedex.json

Example

Request: /createPokemon

Body: {"name":"pichu","imgLink":"https://img.pokemondb.net/artwork/vector/pichu.png","type1":"electric","type2":"","height":"0.3","weight":"2.0","hp":"20","atk":"40", ...}

Result: {"id":152,"num":"152","name":"pichu","type":["electric"],"height":"0.3 m","weight":"2.0 kg","stats":{"hp":"20","atk":"40", ...},"img":"https://img.pokemondb.net/artwork/vector/pichu.png"}

Create Type

Function

Creates a type with the specified parameters.

Parameters

name (string, required): The name of your type

weaknesses (string[], optional): A list of the other types your type is weak to. Accepts "itself" as an option for self-weak types.

resistances (string[], optional): A list of the other types your type is resistant to. Accepts "itself" as an option for self-resistant types.

immunities (string[], optional): A list of the other types your type is immune to. Accepts "itself" as an option for self-immune types.

Return Format

A JSON object matching the data found in types.json

Example

Request: /createPokemon

Body: {"name":"dark","weaknesses":["fighting","bug"],"resistances":["ghost","itself"],"immunities":["psychic"]}

Result: {"name":"dark","weaknesses":["fighting","bug"],"resistances":["ghost","dark"],"immunities":["psychic"]}

Edit a Pokemon's Type

Function

Parameters

type1 (string, required): The first type you want the Pokemon to have

type2 (string, optional): The second type you want the Pokemon to have

Return Format

Example

Request: /createPokemon
